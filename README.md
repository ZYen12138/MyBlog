## 介绍

这是一个用Flask实现的一个小博客:cherry_blossom:，目前还在开发过程中，更新时间不详...

## 启动

第:one:次启动，需要初始化数据库：

```bash
python app.py init
python app.py migrate
python app.py upgrade
```

之后运行服务器：

```bash
python app.py runserver
```

访问localhost:5000即可
